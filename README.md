# motobot

**A modular IRC conversation bot written in C.**

Based on [rochako](https://github.com/rochack/rochako)

# Features
- connects to irc
- responds using markov chains
- reads and writes channel activity to a log file

# Installation
- run `make`
- make a config file
- run the program
